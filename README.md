# 💪 筋力アップトレーニング管理アプリ v2.0

自分専用のトレーニング内容をカスタマイズできるPWAアプリです。

## 🌟 主な機能

### ✨ NEW! トレーニング内容のカスタマイズ
- **完全カスタマイズ可能** - 各曜日のトレーニング内容を自由に編集
- **エクササイズの追加・削除** - 必要なエクササイズだけを管理
- **タイトルとバッジの変更** - あなたのスタイルに合わせて
- **デフォルトに戻す** - いつでも初期設定に戻せます

### 📊 進捗管理
- ✅ **今日のトレーニング表示** - その日のメニューを自動表示
- 📈 **週ごとの達成率** - 自動計算で進捗を可視化
- 📜 **履歴記録** - 過去の週のデータを保存

### 📸 記録機能
- **写真記録** - ビフォー/アフター写真を保存
- **データエクスポート/インポート** - バックアップと復元が可能

### 🔔 その他の機能
- **通知機能** - 設定した時刻にリマインダー
- **PWA対応** - ホーム画面に追加してアプリとして使用可能
- **オフライン動作** - ネット接続なしでも使用可能
- **データ永続化** - すべてブラウザに保存

## 📲 使い方

### 初回セットアップ
1. アプリを開く
2. 「編集」タブでトレーニング内容をカスタマイズ
3. 設定タブで通知を有効化（任意）

### 日々の使い方
1. 「今日」タブで今日のトレーニングを確認
2. 完了したエクササイズをタップしてチェック
3. 自動的に進捗が保存されます

### トレーニング内容の編集
1. 「編集」タブを開く
2. 編集したい曜日の「編集」ボタンをクリック
3. タイトル、バッジ、ポイントを変更
4. エクササイズを追加・削除・編集
5. 「保存」をクリック

### データのバックアップ
1. 「設定」タブを開く
2. 「データをエクスポート」をクリック
3. JSONファイルがダウンロードされます

### データの復元
1. 「設定」タブを開く
2. 「データをインポート」をクリック
3. エクスポートしたJSONファイルを選択

## 🏋️ デフォルトトレーニングスケジュール

アプリには以下のデフォルトスケジュールが含まれています（自由にカスタマイズ可能）：

- **月曜日**: 胸・三頭筋（ジム）
- **火曜日**: 脚・臀部（ジム）
- **水曜日**: アクティブレスト（自宅）
- **木曜日**: 背中・二頭筋（ジム）
- **金曜日**: 肩・腕（ジム）
- **土曜日**: 全身 or 体幹（選択可）
- **日曜日**: 完全休養

## 💾 データ保存について

すべてのデータはブラウザのLocalStorageに保存されます：
- トレーニング内容のカスタマイズ
- 日々の進捗記録
- 過去の履歴
- ビフォー/アフター写真
- 各種設定

⚠️ **注意**: ブラウザのデータを削除すると、すべてのデータが失われます。定期的にエクスポート機能でバックアップを取ることをおすすめします。

## 📱 インストール方法

### iOS (Safari)
1. Safariでアプリを開く
2. 画面下部の共有ボタンをタップ
3. 「ホーム画面に追加」を選択

### Android (Chrome)
1. Chromeでアプリを開く
2. メニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択

または、アプリ内の「設定」→「ホーム画面に追加」をクリック

## 💡 Tips

- **週の途中で内容を変更しても大丈夫**: 既存の進捗データは保持されます
- **バックアップは定期的に**: 大きな変更の前にはデータをエクスポートしましょう
- **写真は軽量化を**: 大きな画像はブラウザの容量を圧迫する可能性があります
- **通知の活用**: トレーニングを習慣化するために通知機能を使いましょう
