# 💪 筋力アップトレーニング管理アプリ

週5-6日のジムメイン筋力トレーニングを管理するPWAアプリです。

## 🌟 機能

- ✅ **今日のトレーニング表示** - その日のメニューを自動表示
- 📊 **進捗管理** - 週ごとの達成率を自動計算
- 📈 **履歴記録** - 過去の週のデータを保存
- 📸 **写真記録** - ビフォー/アフター写真を保存
- 🔔 **通知機能** - 設定した時刻にリマインダー
- 📱 **PWA対応** - ホーム画面に追加してアプリとして使用可能
- 💾 **オフライン動作** - ネット接続なしでも使用可能

## 📲 使い方

1. アプリを開く
2. 設定タブで通知を有効化（任意）
3. 今日のトレーニングを確認
4. 完了したエクササイズをタップ
5. 自動的に進捗が保存されます

## 🏋️ トレーニングスケジュール

- **月曜日**: 胸・三頭筋（ジム）
- **火曜日**: 脚・臀部（ジム）
- **水曜日**: アクティブレスト（自宅）
- **木曜日**: 背中・二頭筋（ジム）
- **金曜日**: 肩・腕（ジム）
- **土曜日**: 全身 or 体幹（選択可）
- **日曜日**: 完全休養
